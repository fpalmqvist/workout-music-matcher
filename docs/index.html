About This Project

Workout Music Matcher is an Android application that intelligently matches Spotify tracks to your workout blocks based on real BPM (beats per minute) data. Whether you're training for a cycling event, running a specific cadence, or need music that matches your workout intensity, this app finds the perfect songs for every moment.

How It Works

  1. Load Workout: Import your ZWO workout file or select from saved workouts
  2. Choose Playlist Source: Select from your Spotify playlists or use "Liked Songs"
  3. Generate Playlist: The app analyzes workout blocks and matches Spotify tracks based on real BPM data
  4. Play Workout: Control playback with automatic track transitions aligned to your workout timing
